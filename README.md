# Holiday Itinerary / Non-Air Travel Portal

**Holiday Itinerary / Non-Air Travel Portal** is a modular operations and commercial workflow platform for Satguru Travels. It supports non-air tourism query handling, inter-branch collaboration, dynamic masters, user-based routing, SLA-driven escalation, quote building, supplier sourcing, approvals, booking conversion, finance workflows, and operations analytics. The platform is designed to be configuration-driven, API-first, and extensible for future supplier, payment, and enterprise integrations.

## Scope Guardrails

- **Air ticketing is out of scope**.
- Focus exclusively on **Non-Air** services (hotels, packages, visa, car rental, cruise, insurance, MICE, sightseeing, train/rail, transfers, tours, ferries, and extensible future services).
- Query workflow and quote-to-booking workflow are both required, but modeled as separate lifecycles.
